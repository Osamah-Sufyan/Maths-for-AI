# 🧠 Mathematics Roadmap for Mastering AI & Machine Learning

This repository contains a **math-first roadmap** for mastering **Artificial Intelligence (AI)** and **Machine Learning (ML)**.

The goal is **deep understanding**, not just the ability to use frameworks like PyTorch or TensorFlow.

> If you understand the math, the code becomes trivial.  
> If you don’t, the code becomes magic.

---

## 📌 Philosophy

- Focus on **why algorithms work**, not only how to run them
- Learn math **as it appears in ML**, not abstractly
- Separate **core pillars** from **advanced / research topics**
- Use **theory + intuition + practice**

---

# 📐 Pillar 1: Linear Algebra (The Language of ML)

### Topics
- Vectors, matrices, tensors
- Vector spaces, basis, dimension
- Matrix multiplication & transformations
- Rank, inverse, transpose
- Orthogonality & projections
- Eigenvalues & eigenvectors
- Matrix decompositions (SVD, eigendecomposition)
- Tensors & broadcasting
- Numerical linear algebra

### ML Applications
- Neural network layers
- PCA & dimensionality reduction
- Embeddings
- Attention mechanisms
- Stability of training

### 📚 Recommended Books
- **Gilbert Strang** – *Linear Algebra and Its Applications*
- **Sheldon Axler** – *Linear Algebra Done Right*
- **Trefethen & Bau** – *Numerical Linear Algebra*

### 🎥 YouTube
- MIT OpenCourseWare – *Linear Algebra (Gilbert Strang)*
- 3Blue1Brown – *Essence of Linear Algebra*
- Steve Brunton – *Applied Linear Algebra*

### Topics

- Expectation, variance, covariance
- Joint, marginal & conditional distributions

- Maximum Likelihood (MLE) & MAP
- Hypothesis testing
- Information-theoretic quantities
- Stochastic processes & Markov chains

### ML Applications
- Loss functions
- Bayesian models
- Generative models
- Uncertainty estimation
- Reinforcement learning

### 📚 Recommended Books

- **Bishop** – *Pattern Recognition and Machine Learning*
- **Casella & Berger** – *Statistical Inference*
- **DeGroot & Schervish** – *Probability and Statistics*

### 🎥 YouTube

- MIT OpenCourseWare – *Probability*
- Brandon Foltz – *Statistics & Probability*
